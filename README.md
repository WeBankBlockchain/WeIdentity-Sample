## 整体介绍

weid-sample是基于 [WeIdentity](https://weidentity.readthedocs.io/zh_CN/latest/README.html)开发的样例程序，提供了一整套的流程演示，可以帮您快速理解WeIdentity的运行机制，您也可以参考该样例程序，开发您的WeIdentity应用。
